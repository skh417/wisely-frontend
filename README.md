## 프로젝트 주요 목적
- 실제 협업시 사용하는 Scrum, Sprint Meeting, Stend Up Meeting 방식 경험.
- FrontEnd / BackEnd 협업을 통한 커뮤니케이션 및 협동 작업의 중요성 이해.
- 위 홈페이지에 대한 다양한 UI 및 디자인 요소, 애니메이션 등을 React/styled-components 로 구현.
- 제작 과정에 있어 사용자 중심 구조에 대한 이해도 향상.
